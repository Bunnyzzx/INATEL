- Uso da IA no desenvolvimento das atividades
Durante a realização dos exercícios de Programação Orientada a Objetos em Java, 
utilizei o ChatGPT como ferramenta de apoio para esclarecer dúvidas conceituais,
corrigir erros de sintaxe e auxiliar no entendimento da lógica das implementações.

- Como a IA auxiliou
explicando conceitos de forma gradual;
mostrando diferenças entre abstração e sobrescrita;
esclarecendo o funcionamento de getters e setters;
explicando a lógica do for-each;
ajudando a entender o uso de ArrayList e HashMap;
corrigindo pequenos erros de sintaxe;
auxiliando no entendimento das relações entre classes.

- Dificuldades encontradas
entender quando utilizar classe abstrata;
compreender a diferença entre composição e agregação;
interpretar corretamente o uso de interfaces;
adaptar conceitos de Java vindo do C++;
entender a sintaxe do for-each e estruturas do Java.

- O que aprendi
como utilizar herança com extends;
como sobrescrever métodos utilizando @Override;
quando utilizar métodos abstratos;
a função dos getters e setters;
a diferença entre composição e agregação;
como usar ArrayList para armazenar objetos;
como utilizar HashMap para impedir duplicidade de entidades;
como interfaces funcionam com implements;
como aplicar polimorfismo na prática.
